# Quiz App
 A quiz app of code related questions within time limit. In which incorrect answer will deduct the user time by ten seconds. At the end of the quiz, what is left of the user time will be recorded as the score. 

 Authentication and authorization is being implemented to ensure only registered user have access to the quiz. And user data is stored in the database.
